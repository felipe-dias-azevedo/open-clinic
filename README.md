# open-clinic
Open Source Clinic System focused on Management of Patients

## Architecture
* Node.JS       -> Server
* Python GTK+   -> Linux Client
* ???           -> Web Client
* ElectronJS    -> Windows, MacOS, Linux (Secondary) Client

## Run
### Server

```cd server/```

```npm i```

```npm start```

### GTK3+ Linux Client

```cd linux/```

```pip install -r requirements.txt```

```python3 start.py```
